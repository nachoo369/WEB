Alojado en localhost::MAMP
Instalación de MAMP [text](https://www.mamp.info/en/downloads/)
--------------------------------------COMO USAR------------------------------------------------------------
1.Descargar e instalar MAMP
2.Dirigirse a C:\MAMP\htdocs
3.Importante tener entorno de trabajo en htdocs ubicada en carpeta MAMP
4.Inicie el servicio MAMP
5.Entre a localhost desde navegador.
